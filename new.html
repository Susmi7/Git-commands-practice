Introduction

  1. Python is an interpreted, high-level, general-purpose programming language.
  2. Created by Guido van Rossum
  3. first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant whitespace.
  4. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects

Prerequisite

  1. You need to install Python
  2. Add python to your Environment variables

Note:

  1. All the code discussed in this tutorial uses python 3.7.4.
