Introduction

Why Git?

  1. Git is a version control tool.
  2. Git keep track of changes to code
  3. Git helps in synchronizing code between different people
  4. It helps in testing differt versions of the code without losing original one.

Prerequisite

  1. You need to create a GitHub account.
  2. You need to install git in your computer.

Note:

  1. All the commands discussed are used from terminal or command prompt.
